# 大学生生活网

一个为大学生提供投稿和内容分享的网站平台。

## 项目简介

大学生生活网是一个专注于大学生投稿服务的平台，提供学习经验、社团活动、志愿服务、社会实践、体育健康、劳动教育、德育教育等多个领域的文章分享。

## 功能特点

- 📝 文章投稿系统
- 📚 文章分类浏览
- 🔍 文章搜索功能
- 💬 评论互动系统
- 📱 响应式设计
- 🎨 现代化UI界面

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- 响应式设计

## 部署

本项目可以部署到以下平台：

### Netlify 部署

1. 将代码推送到 GitHub 仓库
2. 在 Netlify 中导入该仓库
3. Netlify 会自动构建和部署

### 本地运行

使用 Python HTTP 服务器：

```bash
python -m http.server 8000
```

然后在浏览器中访问 `http://localhost:8000`
